# 🧘 Yoga Classes Registration System

A modern and responsive **Yoga Classes Registration System** developed using **PHP, MySQL, HTML, CSS, and JavaScript**. This web application enables users to register, log in securely, book yoga classes, explore yoga programs, manage bookings, and monitor their wellness journey through an interactive dashboard.

---

## 📌 Project Overview

The Yoga Classes Registration System is designed to simplify the management of yoga classes for both users and administrators. It provides a clean, user-friendly interface with secure authentication and an intuitive dashboard for a seamless user experience.

---

## ✨ Features

- 🔐 Secure User Registration & Login
- 🔑 Password Hashing Authentication

## 🚀 Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/YourUsername/yoga-classes-registration-system.git
```

### 2️⃣ Move Project

Copy the project folder into:

```
xampp/htdocs/
```

### 3️⃣ Start XAMPP

Start:

- Apache
- MySQL

### 4️⃣ Create Database

Open:

```
http://localhost/phpmyadmin
```

Create a database named:

```
yoga_db
```

Import your SQL database file.

### 5️⃣ Run Project

Open:

```
http://localhost/Yoga_Project/register.php
```
